Adds a Bitcoin Donate Button widget (with QR-code in a lightbox). If you have not used Bitcoin before, you might want learn a little about it first.

== Description ==

This widget will add a image button to your pages, posts in order that the user can donate bitcoin or altcoins to your wallet.

== Installation ==

1. Upload plugin-name.php to the /wp-content/plugins/ directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. From Wordpress menu->Appearance->widgets drag&drop widget to widget area
1. Set widget title and wallet address and click save. That's all.
